# Thunder Storm (Han Solo)
platform : Spectrum 128KB
made with AGD

A small game with Han Solo for you spectrum.

Your Mission si to rescue the princess leia from Dath Vader.

Keys:
Sinclair Joystick 1

Keyboard: 
O -> Left
P -> Right
Q -> Jump
